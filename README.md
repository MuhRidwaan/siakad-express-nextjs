# siakad-express-nextjs

Aplikasi Sistem Informasi Akademik (SIAKAD) Full-Stack.

## Tech Stack
- **Backend**: Express.js (TypeScript), MySQL (`mysql2`), JWT Auth, bcrypt, multer, `tsx`
- **Frontend**: Next.js (App Router), TypeScript, Bootstrap 5 & Bootstrap Icons, Context API

## Fitur Utama
- **Authentication**: Login & Register dengan Hashing Password (`bcrypt`) dan Token (`JWT`).
- **Role-based Authorization**: Multi-role pengguna (`Admin`, `Operator`, `Viewer`).
- **Manajemen Mahasiswa**: CRUD Data Mahasiswa, Upload Foto Profil, Filter Prodi/Angkatan, Pencarian, & Paginasi.
- **Manajemen Program Studi**: CRUD Data Program Studi.
- **Manajemen User**: Pengelolaan akun & fitur Reset Password acak (Khusus Admin).
